Modern Arcade Guides (Design #1)

Visit Website

Project Details

This site was targeted towards younger teenage boys who enjoy gaming. Going through the design process with prototyping and then actually coding this site ended up taking me about 35 hours.

Purpose and Scope

I made this site to put some of my knowledge of arcade gaming into a space that could be shared with others so they could hopefully improve and experience more success. Coding this website took me a while as I ended up creating five pages to fit the content I wanted.

Design Decisions

The main descriptor I had on my mind while designing this sight was, “welcoming.” As a result of this I decided to implement some fun colors that give off a very cheerful and inviting energy. I also wanted a pretty simplistic layout to again make it easy to navigate and more welcoming.

Required Elements

  • Bootstrap table
  • Bootstrap form
  • Bootstrap lists
  • General use of Bootstrap throughout the design

Results

In the end I was able to create a simple and inviting website for providing people with several modern arcade guides. I learned to utilize Bootstrap effectively to implement helpful elements and save myself valuable time in the coding process. By following the necessary guidelines of the assignment and creating a nice finished product I was given an A grade for this project.
