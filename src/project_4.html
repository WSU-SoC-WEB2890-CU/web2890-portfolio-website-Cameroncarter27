Recreation of Tommy Haas Website

Visit Website

Project Details

For this project I had to choose an existing site and try to recreate it's visual style as closely as I could. This project ended up taking me about 5 hours to complete.

Purpose and Scope

The purpose of this project was to test our knowledge of styling HTML to get a desired look that was as close as possible to the original. This project was only a single page.

Design Decisions

All of the design decisions that I made were made in the interest of looking as close to the original product as possible.

Required Elements

This project required me to utilize my knowledge of HTML and CSS to find the best way to accurately include all of the visual elements that the original page had.
