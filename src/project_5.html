Sunset Animal Shelter Website

Project Details

This project was made for a class assignment. For this assignment we were tasked with creating our own site for a shelter that we made up. Making this site from scratch ended up taking me about 25 hours to complete.Â 

Purpose and Scope

Ultimately the purpose of this project was to fulfill the requirements of the assignment. However, if this site was launched with more of an intention of being a public site for people to use then the purpose would be to promote this shelter. This site ended up being 3 pages.

Design Decisions

I really enjoy sunsets and so I decided to implement sunsets into the design of this project. This gave me a good color pallet as well as a unique background image that became the basis of all of the pages. I also wanted it to be a more fun and lighthearted site for a shelter so I included fun pictures of the pets such as the one of my sister's dog with sunglasses on on the home page.

Required Elements

  • Background image
  • Images in the main content
  • Form
  • Table
  • Links
  • Logo

Results

This project helped me to be much more confident in my website making skills as it is one of my first projects to really start looking like a modern site. While it looks a little dated at parts I am still overall very proud of it still as I think it looks pretty fun and it did receive an A grade.
