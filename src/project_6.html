My First Portfolio Website

Project Details

This class project had me make my very first portfolio website. By the time I finished this project I had spent about 20 hours making it.

Purpose and Scope

The purpose of this project was to give me experience with making a multi-page website and to put my knowledge I had learned throughout the class to the test as this was the final project of this particular course. This was my very first 4 page website so this was particularly challenging for me at the time.

Design Decisions

The main purple of this website was chosen to signify Weber State University as this assignment was supposed to be Weber themed. I then chose yellow as an accent as it’s the complementary color of purple and I wanted some other color besides just purple. We had also learned about background graphics and thus I implemented one to add some visual flair.

Required Elements

As this class had recently taught me about background graphics I was required to implement one into this project. I also had to put in a video that related to me in some way and then a form for people to contact me.
  • Background graphic
  • Image
  • Video
  • Form

Results

With this project I ended up with a fun finished product that had allowed me to play around with certain elements and tested my then fairly limited knowledge of CSS and HTML. I still enjoy the background graphic quite a bit and with the rest of the design it’s still fun to look back and see how far I’ve come. This project was a success in my eyes as tested me and I received an A grade on it.
